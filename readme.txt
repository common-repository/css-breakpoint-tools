Set CSS styles for different devices easily.

== Description ==

Set CSS styles for 4 different devices (small phones, phones, tablets, desktops) easily:

* Extra small devices (portrait phones, less than 576px)
* Small devices (landscape phones, less than 768px)
* Medium devices (tablets, less than 992px)
* Large devices (desktops, less than 1200px)

PS: The CSS styles inserted in this plugin will be loaded after the others (themes, plugins) and should therefore overwrite them.

== Installation ==

Upload the CSS Breakpoint plugin to your blog, Activate it.

1, 2, 3: You're done!

== Frequently Asked Questions ==

= Can I reduce the size of the logo for mobile devices? =

Yes sure! Just enter the styles in the "768px" section.

= Can I increase the size of the post title for desktops? =

Yes sure! Just enter the styles in the "1200px" section.
